# Avnet MicroZed board definition

This directory vendors only `microzed_7020/1.4` from the official Avnet `bdf`
repository, fixed at commit:

`e9723b55a68b106fb4cdb645974d096cd3594a2a`

The upstream license is retained at `microzed_7020/1.4/LICENSE`. Vivado uses
this directory through the project-local `board.repoPaths` setting; no files
need to be copied into the Vivado installation.
